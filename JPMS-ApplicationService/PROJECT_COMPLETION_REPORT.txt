================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

Common Issues & Solutions:

Service Won't Register with Eureka:
  - Ensure Eureka Server started first
  - Check service-url in application.properties
  - Look for "Registering instance" log message
  - Wait 10-20 seconds for registration

JWT Token Errors:
  - Ensure same JWT secret in all services
  - Check token format: "Bearer <token>"
  - Verify token not expired (24-hour expiration)
  - Ensure token from login endpoint

Database Connection Issues:
  - Verify MySQL running: mysql -u root -proot
  - Check databases exist: SHOW DATABASES;
  - Verify connection string in application.properties
  - Check MySQL user permissions

Gateway Routing Not Working:
  - Verify service name matches Eureka registration
  - Service names are case-sensitive
  - Check application.yml gateway routes
  - Ensure service is UP in Eureka first

Build Failures:
  - Run: mvn clean
  - Delete .m2 cache if needed
  - Ensure Java 17+ installed
  - Check Maven 3.6.0+
